# Garmin Exercise Reference

Lookup material for `strength.exercise_name`. Garmin matches names against a catalog of 1,624 exercises across 49 categories. **These same names resolve for Hevy** (spaces to underscores, UPPERCASE resolves to Hevy template IDs). For all other providers that accept exercise names, the value is displayed as Title Case With Spaces regardless of input.

Name lookup is flexible: `BARBELL_BENCH_PRESS`, `barbell_bench_press`, `Barbell Bench Press`, and `barbell bench press` all normalize to `BARBELL_BENCH_PRESS`. Normalization: replace spaces with underscores, uppercase, then look up. If not found, the exercise name and category are dropped from the Garmin workout, weight is preserved, the step is still created, and a coercion warning is returned. See references/exercise-reference.md for normalization and Hevy alias details.

Live docs: https://docs.tryterra.co/planned-workouts-api/overview/garmin-exercise-reference
